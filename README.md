# GPR Blocks

WordPress plugin exclusively developed for the **GPR Empreendimentos** website that can be accessed [here](http://gprinvestimentos.com.br/).

Based on 
- OOP **PHP** for WordPress
- **JavaScript** 
- **Bootstrap** for frontend
